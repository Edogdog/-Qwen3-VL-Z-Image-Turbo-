### 2.2 详细步骤

#### 第一步：上传风格参考图

1. 点击左侧 "Upload Style Reference" 上传区域
2. 选择一张风格参考图片（支持 JPG、PNG 格式）
3. 图片可以是：油画、水墨画、摄影作品、插画等

#### 第二步：分析风格特征

1. 点击 "Analyze Style" 按钮
2. 系统使用 Qwen3-VL 分析图片风格
3. 分析结果会显示在右侧 "Style Analysis Result" 文本框中
4. 分析内容包括：
   - 色彩风格（色调、饱和度、对比度）
   - 氛围情感（情绪、氛围）
   - 构图特点（布局、视角）
   - 艺术风格（流派、表现手法）
   - 视觉元素（物体、纹理、质感）

#### 第三步：输入内容描述

1. 在 "Enter content description" 文本框中输入想要生成的内容
2. 例如：
   - "一位优雅的中国古典美女"
   - "日出时分的山川风景"
   - "一只可爱的猫咪"

#### 第四步：生成风格化图像

1. 点击 "Generate Image" 按钮
2. 系统结合内容描述和风格分析结果生成图像
3. 生成完成后，图像显示在右侧 "Generated Image" 区域

## 三、功能说明

### 3.1 风格分析（VLM）

**输入**：风格参考图片

**输出**：结构化的风格描述文本

**示例输出**：
```
古典东方美学风格，浓郁的中国传统色彩（红、金、黑），
工笔画细腻线条，仕女画优雅仪态，丝帛质感，古典家具背景，
宁静雅致的氛围。
```

### 3.2 图像生成（Z-Image-Turbo）

**输入**：
- 内容提示词
- 风格描述（可选）

**输出**：512x512 PNG 格式图像

**参数说明**：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| num_inference_steps | 9 | 推理步数，越多越精细但越慢 |
| seed | 随机 | 随机种子，用于复现 |

## 五、命令行使用（非界面）

### 5.1 纯图像生成

```python
import sys
sys.path.insert(0, 'modelscope-workshop/lab5-style-transfer')

from gradio_helper import ZImageTurboOV

# 加载模型
generator = ZImageTurboOV('Z-Image-Turbo-int4-ov', device='CPU')

# 生成图像 (使用 generate_image 方法)
image = generator.generate_image(
    prompt='A beautiful Chinese classical beauty in traditional Hanfu',
    height=512,
    width=512,
    num_inference_steps=9,
    seed=42
)

# 保存图像
image.save('output.png')
```

### 5.2 风格分析

```python
import sys
sys.path.insert(0, 'modelscope-workshop/lab5-style-transfer')

from gradio_helper import analyze_style
from optimum.intel.openvino import OVModelForVisualCausalLM
from transformers import AutoProcessor
import json
from pathlib import Path

# 加载 VLM
vlm_model_dir = Path('Qwen3-VL-4B-Instruct-int4-ov')
with open(vlm_model_dir / 'config.json') as f:
    config = json.load(f)
config['vision_config']['embed_dim'] = config['vision_config']['hidden_size']
with open(vlm_model_dir / 'config.json', 'w') as f:
    json.dump(config, f, indent=2)

from optimum.intel.openvino.modeling_visual_language import MODEL_TYPE_TO_CLS_MAPPING, _OVQwen2VLForCausalLM
MODEL_TYPE_TO_CLS_MAPPING['qwen3_vl'] = _OVQwen2VLForCausalLM

vlm_model = OVModelForVisualCausalLM.from_pretrained(vlm_model_dir, device='CPU')
vlm_processor = AutoProcessor.from_pretrained(vlm_model_dir, min_pixels=256*28*28, max_pixels=1280*28*28)

# 分析风格
style_description = analyze_style(vlm_model, vlm_processor, 'style_image.jpg')
print(style_description)
```

## 六、常见问题

### Q1: 风格分析显示错误？

**A**: 检查是否正确上传了图片，确保图片格式为 JPG 或 PNG。

### Q2: 图像生成失败？

**A**: 检查内容描述是否为空，确保输入了有效的内容提示词。

### Q3: 生成速度慢？

**A**: 图像生成基于 CPU 推理，9 步约需 10-15 分钟属正常现象。可以减少 `num_inference_steps` 参数（如设为 4-5 步）以提高速度，但会影响图像质量。

### Q4: 生成的图像是马赛克？

**A**: 这是 VAE 缩放因子未正确应用导致的 bug，请确保代码已更新至最新版本。

### Q5: 如何复现相同结果？

**A**: 在 `generate_image` 中设置固定的 `seed` 值（如 `seed=42`）。
